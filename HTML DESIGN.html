HTML 1.0 (1993)

نسخه ابتدایی با ساختار بسیار ساده و بدون استایل.

<html>
  <head><title>صفحه ساده</title></head>
  <body>
    <h1>سلام دنیا</h1>
    <p>این یک پاراگراف است.</p>
  </body>
</html>

HTML 2.0 (1995)

اضافه شدن فرم‌ها و جدول‌ها برای تعامل بیشتر.

<form>
  نام: <input type="text" name="name">
  <input type="submit" value="ارسال">
</form>

HTML 3.2 (1997)

ورود CSS و جاوااسکریپت به دنیای HTML.

<style>
  h1 { color: red; }
</style>
<script>
  alert("سلام!");
</script>

HTML 4.01 (1999)

ساختاردهی بهتر، پشتیبانی از DOM و استانداردسازی بیشتر.

<div id="content">محتوا</div>
<script>
  document.getElementById("content").style.color = "blue";
</script>

HTML5 (2014)

نسخه مدرن با پشتیبانی از ویدیو، canvas و APIهای تعاملی.

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>

<canvas id="myCanvas"></canvas>

HTML6 (مفهومی)

تمرکز بر امنیت، ماژولار بودن و تعامل با APIها.

<module src="component.html"></module>
<secure-form>
  <input type="password">
</secure-form>

HTML7 (آینده‌نگر)

تعامل با هوش مصنوعی، طراحی واکنش‌گرا پیشرفته و کامپوننت‌های هوشمند.

<ai-terminal>
  سلام! چطور می‌تونم کمک‌تون کنم؟
</ai-terminal>

<responsive-grid columns="3">
  <item>مورد اول</item>
</responsive-grid>
