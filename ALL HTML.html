📜 تاریخچه HTML از نسخه 1 تا 7

HTML 1.0

اولین نسخه HTML که در سال 1993 معرفی شد. بسیار ساده و محدود بود.

<html>
  <head><title>صفحه ساده</title></head>
  <body>
    <h1>سلام دنیا</h1>
    <p>این یک پاراگراف است.</p>
  </body>
</html>

HTML 2.0

در سال 1995 منتشر شد و فرم‌ها و جدول‌ها را معرفی کرد.

<form>
  نام: <input type="text" name="name">
  <input type="submit" value="ارسال">
</form>

HTML 3.2

در سال 1997 با پشتیبانی از CSS و اسکریپت‌ها معرفی شد.

<style>
  h1 { color: red; }
</style>
<script>
  alert("سلام!");
</script>

HTML 4.01

در سال 1999 منتشر شد و ساختاردهی بهتر و پشتیبانی از DOM را آورد.

<div id="content">محتوا</div>
<script>
  document.getElementById("content").style.color = "blue";
</script>

HTML5

در سال 2014 به‌طور رسمی معرفی شد. پشتیبانی از ویدیو، canvas، و APIهای جدید.

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>

<canvas id="myCanvas"></canvas>

HTML6 (مفهومی)

نسخه‌ای پیشنهادی با تمرکز بر امنیت، ماژولار بودن، و تعامل با APIها.

<module src="component.html"></module>
<secure-form>
  <input type="password">
</secure-form>

HTML7 (آینده‌نگر)

نسخه‌ای فرضی با قابلیت‌های هوشمند، تعامل با هوش مصنوعی، و طراحی واکنش‌گرا پیشرفته.

<ai-terminal>
  سلام! چطور می‌تونم کمک‌تون کنم؟
</ai-terminal>

<responsive-grid columns="3">
  <item>مورد اول</item>
</responsive-grid>
